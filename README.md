# fitness 앱

자체 제작 신발과 연동되어 좌우 불균형 교정 서비스를 제공하는 앱

**기능**:
- 자체 제작 신발과 블루투스 통신
- 타이머 기능 및 운동 횟수 카운팅
- 월, 주별 운동량 통계
- 보호자 알림 메세지 서비스
